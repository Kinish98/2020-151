Dynamic Exercise Plan

In this section the system gets the user's daily routine details in the user registration stage. By using these details, the system will predict a suitable exerc time schedule for the user. The exercise plans are highly personalized according to user to user. There are mainly more than 10 exercises are used in the mobile software solution. The exercises change according to day today. By using Q-Learning machine learning algorithm, the exercise scheduling functionality update according to user's daily routine. The selected exeercises are listed in the below section.

  • Arm Raises
  • Military Pushups
  • Abdonominal Crunches
  • Russian Twist
  • Leg Raises
  • Cobra Stretch
  • Side Hop
  • Squats
  • Backward Lunge
  • Knee to Chest exercise

The above mentioned exercise are change according to day by day and there are many more exercises attached in the mobile software solution.The main objectives of this functionality is to predict a heghly peronalized exercise plan feature according to the user's fitness test results.

In this exercise prediction functionality, the system requires the user fitness level to predict a dynamic exercise plan for the user. After calculating the fitness level, the exercises schedule will assign for the user according to the user's fitness level value. One exercise schedule will be assign for a day and a new exercise schedule will be assign to other days.
